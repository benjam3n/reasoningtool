# Universal Goal Analysis - Guess Taxonomy

This taxonomy organizes 100,000+ guesses for questions that apply to ANY goal.

## Structure

Each file contains guesses for ONE question category.
Each guess answers ONE question.
Each guess has CRUX marker (HIGH/MED/LOW).

## File Format

Each file follows this structure:

```markdown
# Category: [Name]

## Overview
- Total questions: N
- Total guesses: M
- High-CRUX questions: X

## Questions

### Q1: [Question text]
[D: from dimension X]

| Guess | CRUX | ASSUME RIGHT -> | ASSUME WRONG -> |
|-------|------|-----------------|-----------------|
| ... | HIGH/MED/LOW | ... | ... |

### Q2: [Question text]
...
```

---

## Build Order

1. Start with HIGH-CRUX categories (Causation, Failure, Decision)
2. Then Core Dimensions (WHO/WHAT/WHY/HOW/WHEN)
3. Then supporting categories (Biases, Systems, etc.)
4. Finally meta-categories (Quality, Values)

---

## Linking

Each guess can reference:
- Related questions in other files
- Domain-specific overlays
- Prerequisite questions (must answer Q1 before Q2)
