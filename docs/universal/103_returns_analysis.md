## Key Insight

**VOI ≠ analytical depth. VOI = action divergence.**

"What does each additional unit provide?" sounds precise but has LOW VOI - you still make similar investment decisions.

"Are returns actually diminishing?" has HIGH VOI - if returns are appreciating, you continue rather than stop.

---

## Key Lesson

**The claim "diminishing returns" is often:**
1. A convenient excuse to stop
2. Unexamined assumption from other domains
3. Wrong when errors prove more is needed

**Test**: If errors in the current domain would be prevented by more of the thing in question, returns are APPRECIATING, not diminishing.
